​This repository contains two end-to-end projects: Ford Car Price Prediction and Insurance Cost Prediction. These models were developed following an intensive three-day study focused on mastering the mathematical intuition behind linear regression and efficient implementation techniques.


# ​Key Features
1. ​Pipeline Development: Full end-to-end workflows including EDA, feature engineering, and data preprocessing (Encoding, Scaling, Train-Test Splitting).
2. ​Evaluation: Model performance analyzed using R^2 and Adjusted R^2 metrics.
3. ​Documentation: Detailed insights and handwritten notes are available on my portfolio site: [https://cyraemin.github.io/linear-regression/]


# ​Technical Summary
​The work transitions from foundational statistical theory (cost functions and gradient descent) to practical application, demonstrating that complex machine learning models can be built with clean, concise, and optimized code.

# Technical Stack
 ​Core Data Science
-> ​Language: Python
-> ​Libraries: NumPy, Pandas, Scikit-Learn (SKLearn), Matplotlib, Seaborn
-> ​Methodology: Exploratory Data Analysis (EDA), Feature Engineering, Label & One-Hot Encoding, Standard Scalar Transformation, 
  Matrix Operations, and Train-Test Splitting.
